## ⚙️ Installation Guide

Follow these steps to get the Endless War DLC running on your machine. 

1. **Locate your HOI4 Directory:** Find the root folder where your HOI4 dlc folder 📁 is located.  NOT THE DLC DATA 
2. **Download the DLC:** Grab the latest `dlc.zip` from the [Releases](../../releases) tab on this GitHub page.
3. **Extract the Files:** Unzip the downloaded file. 
4. **Drag and Drop:** Move the `Dlc` folder directly into your game's `dlc` directory.
5.  If it ask you to 'Replace folder in place' do it won't break your game I promise.

6. You need to run CreamAPI. It is not hard; you just select Hoi4 and hit the '+' by the game name and '✓'. It does not add DLCs; it makes it so it does not have a red triangle and it works in online. I would not recommend it, as it could get you banned from online. Link: https://github.com/FroggMaster/CreamInstaller

## 🛠️ Troubleshooting & Known Issues

If the DLC isn't loading or the game crashes upon reaching the endless trigger, check the following:

* **Are your files in the right place?** Ensure you didn't accidentally mix up folders; they should be in the DLC folder.
